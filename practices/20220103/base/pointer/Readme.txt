1、指针的作用：通过指针访问内存
    1. 内存编号从0开始， 一般用十六进制表示
    2. 可以使用指针变量保存地址

2、指针变量与普通变量的区别
    1. 普通变量存放数据， 指针变量存放地址
    2. 指针变量可以通过 “*”操作符，操作指针变量指向的内存空间， 这个过程叫解引用

3、指针的使用
    1. 可以通过&符号获取普通变量地址
    2. 可以利用指针记录地址
    3. 对指针变量解引用，可以操作指针指向的内存

4、所有指针类型在32位操作系统下是4个字节

5、空指针与野指针
    5.1、空指针
        1. 空指针：指针变量指向内存种编号为0的内存空间[呢哦村编号为0~255是系统占用内存， 不允许用户访问]
        2. 用途：初始化指针变量
        3. 注意：空指针指向的内存是不可以访问的
     5.2 野指针：指向非法内存空间
     总结：空指针和野指针都不是我们申请的空间，因此不要访问

6、const 关键字的使用[类似java的final]
    1. const 修饰指针，指针指向可以改,指针指向的值不能改
    2. const 修饰常量，指针指向不能修改，指针指向的内存空间数值可以修改
        进行指针数值赋值操作，指针指向的内存空间数值发生变更，内存地址不变
    3. const 修饰常量也修饰指针
    技巧：看const右侧紧跟着的是指针还是常量, 是指针就是常量指针，是常量就是指针常量

7、指针作为函数的参数使用，可以修改指针指向的内存空间对应的数据[可以修改实参的值]
此点需要看视频学习补充理解
