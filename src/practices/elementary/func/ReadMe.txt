4、函数重载注意事项
    1、参数的引用
        1.函数形参：void func(int &a){}; 此种方式可以接收变量：func(a);
        2.函数形参：void func(const int &a){} ； 此形参只能接收常量 : func(10);

     2、函数重载遇到默认参数
        1.void func2(int a, int b = 10) 与 void func2(int a) ; 但执行 func2(10) 函数调用时，编译器无法判断要使用哪个函数，所以会报错; 应该避免此种情况
