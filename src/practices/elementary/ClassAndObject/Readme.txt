一、 类与对象
1、struct和class区别
    1.在C++中 struct和class唯一的区别就在于默认的访问权限不同：
        struct 默认权限为公共
        class   默认权限为私有

2、类成员属性设置为私有
    优点1：将所有成员属性设置为私有，可以自己控制读写权限
    优点2：对于写权限，我们可以检测数据的有效性

3、让一个类作为另外一个类的成员（看 CirleAndPoint.cpp）

4、构造函数调用规则
    4.1 如果用户定义有参构造函数，c++不在提供默认无参构造，但是会提供默认拷贝构造
    4.2 如果用户定义拷贝构造函数，c++不会再提供其他构造函数
    4.3 构造函数的使用
        1、两种分类方式：
            按参数分为： 有参构造和无参构造
            按类型分为： 普通构造和拷贝构造
        2、三种调用方式：
             括号法
             显示法
             隐式转换法
    4.4 、两个注意项
        1、不要利用拷贝构造函数初始化匿名对象，编译器认为：Person p = Person p;
        2、调用无参构造函数不能加括号，如果加了编译器认为这是一个函数声明

5、深拷贝与浅拷贝
    4.1 编译器提供的拷贝构造函数，默认是浅拷贝
    4.2 浅拷贝：简单的赋值拷贝操作
    4.3 深拷贝：在堆区申请空间， 在进行拷贝操作
    4.4 当成员属性是其他类对象或指针类型时，用户要自定义拷贝函数（先在堆区开辟空间再赋值[int *a = new int(10);]）; 原因;析构函数会释放对象的指针或者对象地址为null  (讲课老师使用 VS 演示的)
    4.5 当成员属性是其他类对象或指针类型时，可以不自定义拷贝函数；原因：析构函数会释放对象的指针或者对象地址为null或不变（CLion 2018 演示的）

6、初始化列表
    6.1 作用：用于创建对象时，初始化对象属性
    6.2 语法： Person(int a,int b , int c):m_A(a),m_B(b),m_C(c){};

7、类成员变量
    7.1 类对象做类成员变量
    7.2 静态变量做类成员变量
        1、静态成员变量特点
        （1）在编译阶段分配内存
        （2）类内声明，类外初始化
        （3）所有对象共用一份数据
        （4）静态成员变量也有访问权限
        2、 静态成员函数特定说明
         （1）程序共享一个函数
         （2）静态成员函数只能访问静态成员变量 , 非静态成员函数可以调用静态成员变量
         （3）静态成员函数也是有访问权限的

8、const 修饰成员函数与对象 VS static 修饰成员函数
    1、const修饰的对象只能调用const修饰的函数，且const 修饰的函数不能改变变量的值，除非成员函数使用 mutable修饰
    2、普通对象可以调用const 修饰的函数
    2、static 修饰的成员函数，只能调用static 修饰的成员变量或static 修饰的成员函数，static 修饰的变量可以修改
